Gaming AI Assistant
========================

Интерактивный AI-агент‑аналитик для геймеров на Python + React.

## Кратко о проекте

**Gaming AI Assistant** — это BDI‑агент (Belief–Desire–Intention), который:
- отслеживает игры и их цены в реальном времени;
- анализирует тренды и новости игровой индустрии;
- даёт рекомендации по покупкам;
- общается с пользователем в формате диалога.

Архитектура соответствует аннотации:
- **Backend (Python)** — логика BDI‑агента, мониторинг, REST API.
- **Frontend (React)** — панель мониторинга, чат, уведомления.
- **AI‑движок** — интеграция с внешним LLM (например, Claude API) для диалога и рекомендаций.

## Структура (MVP)

Будет создано минимум:
- `requirements.txt` — зависимости Python.
- `backend/` — сервер FastAPI и BDI‑агент.
  - `main.py` — запуск API и фонового мониторинга.
  - `agent/bdi_agent.py` — реализация BDI‑агента.
  - `models.py` — основные Pydantic‑модели (игры, события, сообщения).
- (позже) `frontend/` — React‑клиент (панель, чат, уведомления).

## Как запускать backend (план)

1. Создать виртуальное окружение:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустить сервер:
   ```bash
   uvicorn backend.main:app --reload
   ```

Далее можно будет подключить React‑клиент и реальные игровые API (Steam, IGDB, Metacritic).
